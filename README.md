# rentserve-backend


## 📦 Project Materials and Services rental
This project was done to obtain my bachelor IT degree

## 🚀 Features
- Log in
- Register
- Display materials or services
- CRUD materials or services
- Seed basket
- Pay for basket

## 🛠️ Technologies used
* Java 25 
* **Spring Boot 4.0.1**
* **Gradle**
* Database: postgreSql
* IntelliJ

## ⚙️ Launch of the project
1. Clone the project
> git clone [https://github.com/Br4hms-d3v/rentserve-backend.git](https://github.com/Br4hms-d3v/rentserve)
2. Build and launch your app
> ./gradlew bootRun
3. Check you Swagger
> http://localhost:8080/swagger-ui.html
